AutoFormalizer

Automatically capitalizes messages and adds punctuation to improve readability in chat.

Author: Kai
ID: 1502645455446609951

This plugin formats outgoing messages in real time:

  • Capitalizes sentences automatically
  • Fixes basic grammar structure
  • Adds punctuation if missing
  • Preserves slang like "lol", "idk", "brb"

Install via your client plugin loader using:

plugins/AutoFormalizer Download Manifest
